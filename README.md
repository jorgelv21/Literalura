# Literatura API

Este projeto é uma aplicação Java que consome dados de uma API de livros, listando autores vivos a partir de um ano específico fornecido pelo usuário.

## Requisitos

- Java 17 ou superior
- Maven
- Conexão com a internet para acessar a API
